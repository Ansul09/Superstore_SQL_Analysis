# Superstore_SQL_Analysis
Overview:

  This project analyzes the Sample Superstore dataset using MySQL to uncover insights about sales, profit trends, customer segments, product categories, discount impact, and   regional performance.

Tech Used:
MySQL
SQL (Joins, Aggregations, Window Functions, Date Functions)
GitHub for documentation

Key Business Questions Answered:

What are the total sales and profit?
Which categories and sub-categories perform best?
Which regions/states contribute the most profit/loss?
Which customers are high-value?
How do discounts affect profit?
What monthly/yearly trends appear in sales?
